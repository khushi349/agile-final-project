# agile-final-project


## Overview
This repository contains the final project for the Agile and Scrum course. It demonstrates the use of GitHub Issues, labels, milestones, and project boards to manage product development using Agile practices.

## Features
- Create a product in the catalog
- Retrieve a product from the catalog
- Update a product in the catalog
- Delete a product from the catalog
- List all products
- Search products
- Like a product
- Dislike a product

## Technical Tasks
- Set up CI/CD pipeline
- Write documentation

## Project Management
- User stories created using the format:
  - As a ...
  - I need ...
  - So that ...

- Acceptance criteria follow Gherkin syntax:
  - Given ...
  - When ...
  - Then ...

## Tools Used
- GitHub Issues
- GitHub Labels
- GitHub Milestones
- GitHub Projects
